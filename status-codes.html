Status Code Families

Overview of HTTP 1xx‚Äì5xx status code classes.
Family Meaning Common Examples Developer Action
1xx Informational 100 Continue Proceed as advised by the server.
2xx Success 200 OK, 201 Created Use the returned result; operation succeeded.
3xx Redirection 301 Moved Permanently, 302 Found, 304 Not Modified Follow the Location header and respect caching rules.
4xx Client Error 400 Bad Request, 401 Unauthorized, 403 Forbidden, 404 Not Found Fix the request (URL, parameters, auth, or payload) before retrying.
5xx Server Error 500 Internal Server Error, 502 Bad Gateway, 503 Service Unavailable Investigate server-side issues or retry later with backoff.
