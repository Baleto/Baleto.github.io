Hypertext Transfer Protocol (HTTP)

The blueprint for HTTP was written by Tim Berners-Lee in 1989 as part of the original Web proposal; the first implementation shipped in 1991. HTTP is the application-layer protocol that carries requests and responses between clients and servers.
