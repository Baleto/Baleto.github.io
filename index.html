Hypertext Transfer Protocol (HTTP)

The blueprint for HTTP was written by Tim Berners-Lee in 1989, as part of the initial proposal for the World Wide Web. However, its official release and implementation occurred in 1991. In essence, HTTP serves as the main canal for data transmission, between clients and servers.

Most importantly, HTTP differs from the Hypertext Transfer Protocol Secure (HTTPS), in terms of built-in security capacity.
