SEOs ‚Äî Search Engine Optimization

Search Engine Optimization (SEO) ensures that pages can be discovered, crawled, and indexed correctly. Also that their content matches the searcher's intent. Below is a quick operator board showing how major search engines handle crawl/index directives.

Engine Crawl Agent robots.txt Meta robots Canonical Sitemaps
Google Googlebot Respected. Disallow prevents crawling (not indexing if known via links). index/noindex, follow/nofollow, noarchive supported. Preferred for duplicate control. /sitemap.xml & robots.txt hint.
Bing Bingbot Respected similarly to Google. Same core directives. Also supports max-snippet, etc. Supported. Supported; can submit in Bing Webmaster Tools.
DuckDuckGo (primarily Bing index sources) Respected according to source crawlers. Follows upstream crawler rules. Follows upstream canonical signals. Follows upstream sitemaps.
Applebot Applebot Respected. Core directives supported. Supported. Supported.

Summary. The use of robots.txt is good to control what gets crawled, not what gets indexed; the use of <meta name="robots"> for indexing and link-follow rules on a per-page basis; declaring a <link rel="canonical"> allows to consolidate duplicates; publishing XML sitemaps makes it possible to advertise discoverable URLs. Having clean information architecture and a good content are parts of the foundation.
